Experiments at the Dr. Goard Research lab involve imaging the brains of mice while they play simple video games. The mice have to be trained to play video games, and this is done using an automated sytem that rewards them for their performance.

I redesigned the hardware, software, and electronics of this system. The current system is more reliable, easier to assemble and maintain, and trains mice to acheive higher success rates.

The software was originally written without any game engine. I developed a game engine framework for this project and refactored the code to implement the game engine.
Source code for the project can be found here: Games and Hardware Interfacing, Game Engine
Documentation for the game engine can be found here: Game Engine Docs
A UCSB experiment in the department of anthropology aims to test whether variations in performance between men and women in mental rotation tasks might have been an evolutionary trait that improved hunting ability in early humans. I created a series of video games for this experiment to test participants ability to perform hunting related tasks.

The games can be downloaded here: Windows Installer (x86)
Source code can be found here: Github Link
One of the tasks requires participants to guess the landing location of a projectile.
The participants are also tested in their ability to quickly find camouflaged animals in various photos.
The results are compared to each participant's performance in the mental rotation task.
